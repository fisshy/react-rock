React Rock
=================

Some easy to use tools to generate boilerplates ( work in progress )

Install
-------
With npm: (currently not deployed)

    $ npm install git+https://github.com/fisshy/react-rock.git -g

</ul>

Commands
-------
Generate component boilerplate

    $ rock mc <component>
    
    Generates
    /component
        /actions
            component.js
        /components
            component.jsx
        /constants
            component.js
        /store
            component.js
